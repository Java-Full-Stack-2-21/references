Datatypes:

  • number = numberical values (ex. )
  • string = text (ex. )
  • boolean = true or false values (ex. )
  • null = nothing but not "" or 0 (ex. )
  • undefined = any value declared but not initialized (ex. )
  • object = values of any type that can be placed in key-value pairs (ex. )
  • symbol = primitive type for unique identifers; part of ES6 (ex. )
